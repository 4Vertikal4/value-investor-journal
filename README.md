# Value Investor Journal

Desktop application for long-term value investors.

The project is built primarily for personal portfolio management and investment journaling on Linux.

## Features

* SQLite portfolio database
* Investment journal
* Position management
* Review tracking
* Local-first architecture
* No cloud dependency
